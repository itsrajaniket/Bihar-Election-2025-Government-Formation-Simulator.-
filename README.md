# Bihar-Election-2025-Government-Formation-Simulator.-



Key Features Implemented
Core Functionality:

✅ Display 8 parties with seat badges

✅ Checkbox selection for multiple parties

✅ Live seat calculation and display

✅ Majority mark comparison (122/243 seats)

✅ Clear status messages with color coding

Advanced Features:

✅ Real-time bar chart showing seat distribution

✅ Selected parties highlighted in green in the chart

✅ Smart coalition suggestions (generates valid 3-party combinations)

✅ Click any suggestion to auto-select that coalition

✅ Search/filter parties by name or abbreviation

✅ Dark mode toggle saved in localStorage

✅ Last selection persists when you reload the page

✅ Export results to clipboard

Accessibility:

✅ ARIA labels and live regions for screen readers

✅ Keyboard navigation (focus styles, escape to reset)

✅ Semantic HTML structure

✅ Color contrast meets WCAG standards

User Experience:

✅ Compact, minimal design with few colors

✅ Fully responsive (works on mobile and desktop)

✅ Smooth animations and transitions
